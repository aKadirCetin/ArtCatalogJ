# ArtCatalogJ

ArtCatalogJ is a simple Android application that allows users to select images from their gallery and display them in a list on the main screen.

## Features
- Pick images from gallery
- Display selected images in a RecyclerView
- Simple and clean UI

## Technologies
- Java
- Android SDK
- RecyclerView
- SQLite database

## Installation
1. Clone the repository
2. Open in Android Studio
3. Run the project on an emulator or device
